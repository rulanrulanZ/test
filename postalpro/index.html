PostalPro

PostalPro was a one year long project to redesign RIBBS - USPS National Customer Support Center to create a better experience for business mailers and USPS users. As the UI designer on the project, my mission was to design and build static HTML templates for different page types that would be converted into a drupal theme.

You can visit the original site here and the redesigned site here.

Prime Objective

The primary goal of the redesign was to provide business mailers easy ways to quickly access time-sensitive resources and make the site more accessible to non super users.

I followed the following steps for each template:

  1. Define main goal and template structure
  2. Whiteboard and sketch layout options
  3. Create lo-fi wireframes
  4. Validate with team
  5. Design hi-fi mockup
  6. Share with team, make revisions, repeat
  7. Create architecture map for Drupal integration
  8. Build static template using HTML, CSS, and some javascript
